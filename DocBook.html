Описание

Данное ПО является самостоятельным продуктом.

Данное ПО предсталвяет собой блокнот, с поддержкой семантики языков программирования и тем оформления.

На данный момент поддерживаются следующие ЯП: C++, C, Java, PHP.

Поддерживается взаимодействия с GitHub (отвечает за возможность создавать репозитории, которые будут включать указанные файлы, удалять репозитории, добавлять и удалять новые файлы в существующие репозитории).

Поддерживается создание и сохранение собственных тем оформления.

Примеры использования

Создание блокнота с семантикой

Notebook notebook = new Notebook(); Semantic semantic = new Java(); notebook.setSemantic(semantic); notebook.create("notebook1.txt"); notebook.save();

Создание собственной темы оформления

Notebook notebook = new Notebook(); Theme theme = new Theme(); theme.setTextColor(Color.RED); notebook.setTheme(theme);

Сохранение и загрузка темы

Theme theme = new Theme(); theme.setName("Theme1"); theme.setTextColor(Color.GRAY); theme.save("theme.theme"); Theme newTheme = new Theme(); newTheme.load("theme.theme");
